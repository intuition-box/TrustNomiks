## 4. Aesthetic POV (decisive)

**Dark-first.** Flip the default to dark (keep light as a toggle â€” plumbing already exists via `next-themes`). Dark is the sector default (Dune, Nansen, Arkham, Bubblemaps, Intuition) and the only canvas on which the taxonomy *glows*; the "graph in the void" is the brand. This is the single highest-leverage visual change.

**Color discipline:** chrome stays slate-neutral so the taxonomy carries real semantic weight. Author brand + graph colors in **OKLCH** with a two-tier system (primitive ramps â†’ semantic aliases), adding the missing domain tokens: `--success/warning/info`, `--status-{draft,review,validated}`, `--risk-{low,med,high}`, and `--surface-{1,2,3}` for a real elevation ladder. Give `--primary`/`--secondary` distinct light/dark luminance. Text follows **weight-not-deletion**: metadata at ~50% opacity, full saturation reserved for status, actions, and node colors.

**Type:** load a real typeface (none today). A humanist sans (Inter/Geist family) for UI to de-intimidate DeFi, paired with a **mono for addresses/hashes/triples** and **`tabular-nums` on every numeric** (supply, %, stake) so data scans.
